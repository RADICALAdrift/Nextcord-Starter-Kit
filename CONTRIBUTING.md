# Contributing

Thanks for your interest in contributing!
This project is a simple starter framework for building Discord bots with Nextcord. Contributions that improve clarity, stability, or usability are always welcome.

---

## Ways to Contribute

You can help by:

- Reporting bugs
- Suggesting new features
- Improving documentation
- Fixing issues
- Adding example cogs or improvements to existing ones

---

## Before Opening a Pull Request

Please make sure:

- Your code runs without errors
- You do not include any secrets or tokens
- Changes remain beginner-friendly
- Code follows the existing structure and style where possible

---

## Pull Request Guidelines

When submitting a pull request:

1. Clearly describe what your change does
2. Keep pull requests focused on a single improvement or fix
3. Avoid large unrelated refactors
4. Test your changes when possible

---

## Code Style

This project prefers:

- Clear and readable Python code
- Consistent indentation (4 spaces)
- Simple and understandable logic
- Helpful comments where appropriate

---

## Security

Never commit sensitive information such as:

- `.env` files
- Discord bot tokens
- API keys
- GitHub tokens

---

## Questions

If you're unsure about something, feel free to open an issue first to discuss it.
